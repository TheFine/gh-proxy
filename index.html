GitHub 文件加速

GitHub文件链接带不带协议头都可以，支持release、archive以及文件，右键复制出来的链接都是符合标准的。

release、archive使用cf加速，文件会跳转至JsDelivr

注意，不支持项目文件夹

分支源码：https://github.com/zfb132/WhuHelper/master.zip

release源码：https://github.com/zfb132/WhuHelper/archive/v2.0.0.zip

release文件：https://github.com/zfb132/WhuHelper/releases/download/v2.0.0/app-debug-v2.0.0.apk

分支文件：https://github.com/zfb132/WhuHelper/blob/master/app/build.gradle
